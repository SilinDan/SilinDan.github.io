everyone-can-use-english

 
This article is a note about Xiaolai Li's book 《everyone can use English》.

There is no doubt that the best way to learn English is to use it.


I am practicing this theory these months. 

For example, I replace the browser from baidu to google, and its default language is English. 

So when I encountered  a problem, I have to read English webside instead of Chinese webside.

 The benefit of doing this is that I can read English documentation about program easier than before.

In the future, I think I can do the following things:


- Trying to express my thoughts in English.


- Write article in English and modify my own article.


- collect unintuitive expressions. Sometimes we can't translation Chinese directly in English,so we should collect these sentence, through accumulation to trun these into your own knowledge, eg:


    - 如果我没记错的话……（If my memory serves, …）


    - 我从未想过…… （It never occurred to me that…）


    - 你竟然跟我这么说话！（Are we really having this conversation?）


    - 就知道你有这本事！（Always knew you had it in you!）


    - (这)听着耳熟吧？（Does it ring a bell to you?）


    - 我想不起来那名字了……（That name escapes me….）
